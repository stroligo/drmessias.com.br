# Design System — Dr. José Messias

## Visão da Marca

A identidade visual do Dr. José Messias foi construída para transmitir:

- confiança;
- precisão cirúrgica;
- sofisticação discreta;
- acolhimento humano;
- medicina premium contemporânea.

O branding equilibra:

> excelência técnica + cuidado humano.

---

# Conceito

## Posicionamento

Precisão cirúrgica com cuidado humano.

---

# Logo

## Estrutura

A marca é composta por:

- monograma “jm” manuscrito;
- divisor vertical dourado;
- nome institucional;
- subtítulo médico.

---

## Versão Principal

```txt
jm | DR. JOSÉ MESSIAS
     CIRURGIA GERAL
```

---

## Espaçamento

Área mínima de respiro:

```txt
1x altura do monograma
```

Nunca aproximar elementos da marca.

# UI Style

## Interface

A interface deve seguir:

- muito espaço em branco;
- grid clean;
- poucos elementos;
- visual editorial;
- animações suaves;
- alto refinamento visual.

---

# Fotografia

## Direção

Fotos devem transmitir:

- segurança;
- acolhimento;
- profissionalismo;
- calma;
- sofisticação.

---

## Evitar

- poses agressivas;
- excesso hospitalar;
- iluminação fria;
- fundo poluído;
- estética clínica popular.

---

# Ícones

Estilo recomendado:

- outline fino;
- dourado fosco;
- minimalista;
- geometria limpa.

---

# Estilo Visual

## Deve parecer

- premium;
- contemporâneo;
- médico;
- elegante;
- humano.

---

# Não utilizar

- vermelho médico;
- gradientes exagerados;
- efeitos 3D;
- sombras pesadas;
- azul hospital saturado;
- fontes serifadas clássicas demais.

---
