# drmessias.com.br

Site institucional e presença digital do **Dr. José Messias**, cirurgião geral.

**Domínio:** [drmessias.com.br](https://drmessias.com.br)  
**Repositório:** [github.com/stroligo/drmessias.com.br](https://github.com/stroligo/drmessias.com.br)

---

## Sobre o projeto

Este repositório concentra o desenvolvimento do site profissional do Dr. José Messias: apresentação da especialidade, credenciais, contato e materiais de marca alinhados à identidade visual definida para a prática médica.

O projeto está em fase inicial. Por enquanto, o repositório reúne referências de marca e fotos em `Arquivos/`; a implementação do site será adicionada aqui conforme o escopo for definido.

---

## Identidade visual

A marca trabalha com **azul petróleo / azul escuro** e **dourado**, tipografia sans-serif para o nome e monograma **JM** (duas direções de logo em estudo).

| Pilar | Significado |
|-------|-------------|
| **Confiança** | Segurança e autoridade na relação com o paciente |
| **Experiência** | Trajetória e conhecimento clínico |
| **Precisão** | Técnica e excelência cirúrgica |
| **Humanização** | Acolhimento e proximidade com o paciente |
| **Sofisticação** | Visual elegante, alinhado à medicina de alto padrão |

Referência completa das opções de logo e mockups: `Arquivos/ideias.png`.

## Desenvolvimento local

1. Clone o repositório:

   ```bash
   git clone https://github.com/stroligo/drmessias.com.br.git
   cd drmessias.com.br
   ```

2. Sirva a pasta no ambiente local (ex.: MAMP, Laravel Valet, `php -S`, ou o stack que for adotado para o site).

   O caminho local típico neste workspace:  
   `/Users/gabrielstroligo/Sites/localhost/drmessias.com.br`

Instruções de build, dependências e deploy serão documentadas neste README quando a stack do site estiver definida.
